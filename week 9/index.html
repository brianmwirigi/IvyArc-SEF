WEEK 9 PROJECT

This week, you will be adding RegEX to an HTML form asking potential students of a Software Engineering course for unique information. The form will have the following fields and should fulfil the following conditions:

  1. Get the first name (must contain only alphabets, no digits nor whitespaces)
  2. Get the last name (must contain only alphabets, no digits nor whitespaces)
  3. Get the phone number (must contain only numbers, and must not surpass 11 digits)
  4. Get the email address (must be an email address, i.e., should have the format of “[word].[word]@[word].com”
  5. Get their reason for applying for the course via a textarea (using alphanumeric and whitespace characters, as well as being limited to 300 characters)
  6. Submit button that will check if any of the fields does not match the condition. If even one does not match, the form’s values should not be cleared, and you should use the JavaScript alert() function to inform the user of which field has not matched the conditions.
  7. Cancel button that will clear out the fields of the form
