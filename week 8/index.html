WEEK 7 PROJECT

This week, you will be required to use some of the arithmetic operators that you have studied, the HTML form, and some extra JavaScript functions that you will need to research to create the following:

Using JavaScript and HTML Form, collect two numbers from your user, and have different buttons labelled “Addition”, “Subtraction”, “Division”, and “Multiplication”. Upon clicking each button, your web app should perform the calculation labelled on each button for the two collected numbers.

Finally, display the output of the calculation on the webpage.

Tips: Research the following to help you with the task:

  1. The onclick() function
  2. The Number() function
  3. The innerHTML and value fields of an HTML element
